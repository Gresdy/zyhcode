# 零一万物 Yi (01.AI)

## 1. 概述与发展历程

### 公司背景

零一万物（01.AI）由百度创始人李彦宏创立于 2023 年，是一家专注于人工智能大模型研发的科技公司。团队汇聚了来自国际顶级 AI 研究机构的人才，致力于打造国际一流的大语言模型。

### 发展历程

| 时间 | 里程碑 |
|------|--------|
| 2023 年 6 月 | 公司成立，启动 Yi 系列模型研发 |
| 2023 年 11 月 | Yi-34B 开源，引发关注 |
| 2024 年 | Yi-6B/9B 开源，模型矩阵完善 |
| 2025 年 | Yi-2 开源，国际评测表现出色 |
| 2026 年 | Yi-3 发布，多模态能力增强 |

### 核心产品线

- **Yi 开源系列**：Yi-34B、Yi-9B、Yi-6B 等
- **Yi API 系列**：Yi-Large、Yi-Medium、Yi-Light
- **Yi 多模态**：Yi-Vision、Yi-Doc

## 3. 使用方式详解

### 开源模型本地部署

#### Ollama 部署

```bash
# 拉取模型
ollama pull yi
ollama run yi "你好"

# 指定版本
ollama pull yi:34b
ollama run yi:34b "解释大语言模型"
```

#### vLLM 部署

```bash
# 启动服务
python -m vllm.entrypoints.openai.api_server \
    --model 01-ai/Yi-34B-Chat \
    --tensor-parallel-size 2 \
    --max-model-len 4096

# API 调用
curl -X POST http://localhost:8000/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{
    "model": "01-ai/Yi-34B-Chat",
    "messages": [{"role": "user", "content": "用 Python 写快速排序"}]
  }'
```

### Python API 调用

```python
from openai import OpenAI

# 使用 SiliconFlow 等第三方
client = OpenAI(
    api_key="your-api-key",
    base_url="https://api.siliconflow.cn/v1"
)

response = client.chat.completions.create(
    model="01-ai/Yi-Large",
    messages=[
        {"role": "system", "content": "你是一个专业助手"},
        {"role": "user", "content": "解释什么是 RAG"}
    ],
    temperature=0.7,
    max_tokens=2048
)
print(response.choices[0].message.content)
```

## 5. 优缺点分析

### 优势

1. **开源免费**：Yi 开源模型免费使用，可本地部署
2. **中文能力强**：专为中文优化，中文理解出色
3. **性能出色**：Yi-34B 在多项评测中表现优异
4. **模型矩阵完整**：从 6B 到 34B，多种规格可选
5. **国际评测好**：在国际基准测试中表现突出

### 劣势

1. **商业版价格**：API 服务相比 DeepSeek 偏贵
2. **生态建设**：相比 Qwen、DeepSeek 生态较新
3. **多模态**：多模态能力相比头部厂商有差距
4. **第三方接入**：可用的第三方平台较少

## 8. 最佳实践

### 模型选型

### 开源 vs API

| 场景 | 推荐 | 原因 |
|------|------|------|
| 学习研究 | 开源 6B | 免费、低门槛 |
| 企业部署 | 开源 34B | 数据安全 |
| 快速集成 | API yi-large | 省时省力 |
| 成本优先 | 开源 + 量化 | 极致性价比 |
