# Llama 开源大模型

## 1. 概述与发展历程

### 公司背景

Meta（原 Facebook）AI 研究团队开发的开源大语言模型系列。Llama 是"Large Language Model Meta AI"的缩写，以开源、可商用、可本地部署著称，是开源大模型生态的核心项目。

### 发展历程

| 时间 | 里程碑 |
|------|--------|
| 2023 年 2 月 | Llama 1 发布（7B-65B），开源之始 |
| 2023 年 7 月 | Llama 2 发布，可商用，开源生态爆发 |
| 2024 年 4 月 | Code Llama 发布，专注代码生成 |
| 2024 年 7 月 | Llama 3 发布（8B/70B），性能大幅提升 |
| 2025 年 | Llama 4 发布，多模态支持 |
| 2026 年 | Llama 4.5 发布，推理能力增强 |

### 开源许可

| 版本 | 许可 | 商用 | 权重开放 |
|------|------|------|----------|
| Llama 2 | Llama 2 License | 可商用 | 是 |
| Llama 3 | Llama 3 License | 可商用 | 是 |
| Llama 4 | Llama 4 License | 可商用 | 部分 |

## 3. 使用方式详解

### 本地部署（Ollama）

Ollama 是最简单本地部署方式：

```bash
# 安装 Ollama
# macOS: brew install ollama
# Linux: curl -fsSL https://ollama.com/install.sh | sh

# 拉取模型
ollama pull llama3.2
ollama pull llama3.2:3b
ollama pull llama3.1:405b

# 运行对话
ollama run llama3.2 "解释一下什么是 Transformer 架构"

# 流式输出
ollama run llama3.2 "写一个 Python FastAPI 示例"
```

### Python SDK

```python
import ollama

# 同步调用
response = ollama.chat(
    model='llama3.2',
    messages=[
        {"role": "user", "content": "用 Python 写一个快速排序"}
    ],
    options={
        "temperature": 0.7,
        "num_predict": 2048
    }
)
print(response['message']['content'])
```

### Llama.cpp（高性能推理）

```bash
# 克隆 llama.cpp
git clone https://github.com/ggerganov/llama.cpp.git
cd llama.cpp

# 编译
make

# 下载量化模型（Q4_0 量化）
# 模型转换
python3 convert.py /path/to/llama-3.2-3b/

# 量化
./quantize ./models/llama-3.2-3b/ggml-model-f16.gguf ./models/llama-3.2-3b/ggml-model-q4_0.gguf Q4_0

# 运行
./main -m ./models/llama-3.2-3b/ggml-model-q4_0.gguf -n 2048 -p "解释 Transformer"
```

### vLLM 部署

```bash
# 安装 vLLM
pip install vllm

# 启动服务
python -m vllm.entrypoints.openai.api_server \
    --model meta-llama/Llama-3.1-8B-Instruct \
    --tensor-parallel-size 2 \
    --max-model-len 32768

# API 调用
curl -X POST http://localhost:8000/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{
    "model": "meta-llama/Llama-3.1-8B-Instruct",
    "messages": [{"role": "user", "content": "你好"}]
  }'
```

### HuggingFace Transformers

```python
from transformers import AutoTokenizer, AutoModelForCausalLM

model_name = "meta-llama/Llama-3.2-3B-Instruct"
tokenizer = AutoTokenizer.from_pretrained(model_name)
model = AutoModelForCausalLM.from_pretrained(
    model_name,
    device_map="auto",
    torch_dtype="auto"
)

inputs = tokenizer("解释大语言模型:", return_tensors="pt").to("cuda")
outputs = model.generate(**inputs, max_new_tokens=200)
print(tokenizer.decode(outputs[0], skip_special_tokens=True))
```

## 4. 定价详情

### 开源免费

Llama 是**完全免费**的开源模型，可以：

- 免费下载权重
- 免费本地部署
- 免费商用（需遵守许可协议）

### 部署成本

| 方式 | 硬件要求 | 成本 |
|------|----------|------|
| Llama 3.2 3B | CPU 可运行，GPU 推荐 | 硬件成本 |
| Llama 3.2 8B | 最低 6GB GPU | 硬件成本 |

## 5. 优缺点分析

### 优势

1. **完全免费**：开源可商用，无 API 费用
2. **本地部署**：数据不出本地，隐私安全
3. **灵活定制**：可 fine-tune 专属模型
4. **社区活跃**：丰富的开源生态和工具
5. **量化友好**：多种量化版本，硬件要求低
6. **透明可控**：权重开放，可审计

### 劣势

1. **硬件成本**：需要 GPU 才能高效运行
2. **部署复杂**：相比 API 需要更多技术知识
3. **能力差距**：同规模下能力可能弱于闭源模型
4. **中文能力**：Llama 3 中文训练比例较低
5. **技术支持**：没有官方商业支持

## 7. 多模态能力

Llama 3.2 首次支持多模态：

# 图像理解
ollama run llama3.2-vision "描述这张图片"

import ollama

response = ollama.chat(
    model='llama3.2-vision',

        "role": "user",

print(response['message']['content'])

## 9. Ollama 命令详解

### 常用命令

```bash
# 模型管理
ollama list              # 列出已下载模型
ollama pull llama3.2     # 下载模型
ollama rm llama3.2        # 删除模型
ollama cp llama3.2 my-custom # 复制模型

# 运行
ollama run llama3.2 "你好"           # 直接运行
ollama run llama3.2 --verbose "问题"  # 显示详细信息
ollama run llama3.2 --stream "写诗"   # 流式输出

# 创建自定义模型
ollama create my-model -f Modelfile

# 服务器模式
ollama serve  # 启动 API 服务器 (http://localhost:11434)
```

### Modelfile 示例

```dockerfile
FROM llama3.2

# 设置系统提示
SYSTEM """
你是一个专业的数据分析师，擅长使用 Python 进行数据分析。
"""

# 设置参数
PARAMETER temperature 0.7
PARAMETER top_p 0.9
PARAMETER num_ctx 4096

# 设置 Stop Sequence
PARAMETER stop "用户:"
```
